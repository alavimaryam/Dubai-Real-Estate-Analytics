# Dubai-Real-Estate-Analytics
# Dubai Real Estate Analytics Using Machine Learning

## Research Question

Can machine learning models predict Dubai property prices and identify the main factors influencing property values?

## Objective

This project applies data analytics and machine learning techniques to analyze real estate market data.

## Methodology

The project includes:

- Data cleaning
- Exploratory data analysis
- Feature engineering
- Statistical analysis
- Machine learning modeling
- Model interpretation

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- SQL
- Machine Learning
- Data Visualization

## Expected Outcome

Develop an explainable ML model for property price prediction and market analysis.
